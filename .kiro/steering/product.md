# GeniusQA Product Overview

GeniusQA is a multi-platform system automation and QA platform with three main components:

- **Web Platform**: Modern React web app for product showcase, user authentication, dashboard, and API key management
- **Desktop App**: React Native cross-platform desktop application (Windows/macOS) with native OS automation capabilities
- **Python Core**: Backend service providing system automation, native features, and intelligent processing using FastAPI

The platform enables users to automate system tasks and control native OS features through a unified interface across web and desktop environments.
